Project Overview

An interactive Tic-Tac-Toe web application built with HTML, CSS, and JavaScript. Players can compete against each other or challenge an unbeatable AI opponent powered by the Minimax algorithm.

✨ Features
  • 2 Player mode — play against a friend on the same device
  • vs AI mode — unbeatable AI using Minimax algorithm
  • Scoreboard — tracks X wins, O wins, and Draws across games
  • Winning cells highlighted with green glow animation
  • Draw detection — automatically detected and announced
  • Active player glow — scoreboard shows whose turn it is
  • New Game — fresh round while keeping scores intact
  • Reset Scores — wipes everything and starts fresh
  • Smooth cell pop animation on every move
  • Fully responsive — works on mobile and desktop
🎨 Design
  • Dark theme — deep black background with vibrant accents
  • X = Purple, O = Cyan — clear visual distinction for each player
  • Glowing score boxes highlight the active player's turn
  • Win animation — winning cells scale up with green glow effect
  • Gradient button with purple-to-blue color scheme
